
***COMPILAZIONE***

1) posizionarsi in .../OrderedArray_C
2) make all (nella directory corrente ci deve essere il Makefile opportuno)

***CANCELLAZIONE *.o e *.exe***
1) posizionarsi in .../OrderedArray_C
2) make clean (nella directory corrente ci deve essere il Makefile opportuno)


***ESECUZIONE***

--PER ESEGUIRE ordered_array---
1) Compilare
2) posizionarsi in .../OrderedArray_C/bin
3) ./ordered_array "../../ordered_array_sample_file.csv"

---PER ESEGUIRE ordered_array_c_unit_test---
1) Compilare
2) posizionarsi in .../OrderedArray_C/bin
3) ./ordered_array_tests


COMPILE

    Navigate to .../OrderedArray_C
    Run "make all" (the appropriate Makefile must be in the current directory)

**DELETE .o and .exe

    Navigate to .../OrderedArray_C
    Run "make clean" (the appropriate Makefile must be in the current directory)

EXECUTION

    TO RUN ordered_array

    Compile
    Navigate to .../OrderedArray_C/bin
    Execute "./ordered_array "../../ordered_array_sample_file.csv"

    TO RUN ordered_array_c_unit_test

    Compile
    Navigate to .../OrderedArray_C/bin
    Execute "./ordered_array_tests"
